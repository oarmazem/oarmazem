O Documento HTML

Fazer um site é quase tão simples quanto escrever um texto utilizando o Word.

A página que você visualiza em seu navegador é, basicamente, um simples arquivo texto dizendo ao navegador qual o conteúdo da página, qual sua estrutura e contendo instruções para formatação do layout.

Como curiosidade, acesse agora o site do armazém e clique com o botão direito do mouse em qualquer lugar da página. Irá abrir um menu e então selecione Exibir Código Fonte (ou algo assim). Você verá o código fonte da página. Ou seja, o código que foi escrito para que aquela página fosse exibida.

Está em uma linguagem chamada HTML, bastante fácil de aprender e que não é uma linguagem de programação. É uma linguagem para formatar Hiper Textos. HTML quer dizer Hyper Text Markup Language. Vamos ver como isso funciona:

Abaixo veja a estrutura básica de uma página em HTML


  <!DOCTYPE html>
  <html>

    <head>
    </head>

    <body>
      Você está vendo a estrutura básica de um arquivo HTML
    </body>

  </html>

Este código, no seu navegador, mostraria apenas :

Note estas coisas entre parênteses angulares < >

São as chamadas tags.

Assim <html> é uma tag, <head> é outra tag e <body> é mais uma tag.

A maioria destas tags possui um “fechamento”. Ou seja, uma outra tag iniciando com uma barra inclinada e as tags de abertura e fechamento delimitam o conteúdo ou escopo daquele elemento. Assim </html> fecha a tag <html>. E tudo que está entre a abertura e o fechamento de uma tag, pertence ao escopo desta tag. É o conteúdo dela.

Então note que


  <head>
  </head>

  <body>
    Você está vendo a estrutura básica de um arquivo HTML
  </body>

é o que está dentro de <html> </html>no nosso exemplo.

As tags indicam ao navegador o que são as coisas em um arquivo HTML . As tags <html> e </html>, por exemplo, indicam onde começa e onde termina o documento HTML. A página em si.

E obrigatoriamente há pelo menos duas tags dentro de <html></html>

  1. O elemento <head> (a cabeça ou cabeçalho de uma página HTML)
  2. O elemento <body> (o corpo de uma página HTML)

Entre <head> e </head> colocamos as configurações globais da página.

Entre <body> e </body> vai o conteúdo propriamente dito da página.

O código HTML é estruturado por meio de tags. Elas indicam ao seu navegador o que é um título, o que é um parágrafo, o que é um link, e assim por diante... Aqui você encontra uma lista de todas as tags HTML.
