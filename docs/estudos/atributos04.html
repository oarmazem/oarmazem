Atributos em Tags

Como vimos, tags podem possuir atributos. Tags com e sem fechamento podem ter atributos.

Alguns atributos são específicos para determinadas tags. Enquanto outros são globais e podem ser incluídos em quase todas as tags.

Alguns atributos são opcionais enquanto outros são obrigatórios em determinadas tags.

No artigo anterior vimos como funciona o atributo opcional lang na tag <html>. (<html lang="pt-br">)

O atributo lang está informando ao navegador qual o idioma predominante em todo o documento HTML. Como o escopo da tag <html> é todo o documento, toda a página, consequentemente o atributo lang nesta tag é referente ao idioma do documento.

Porém mesmo que o idioma apresentado na sua página seja o Português do Brasil, ainda assim você poderia ter algum determinado parágrafo em Inglês (ou qualquer outro idioma). Neste caso você ainda poderia informar isto ao navegador inserindo o atributo lang exatamente neste parágrafo cujo idioma não fosse o Português do Brasil.

Vejamos um exemplo:


  <!DOCTYPE html>
  
  <html lang="pt-br">
    
  <head> 
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Exemplificando Atributos</title>
  </head>
  
  <body> 
    <h1>Armazém Café e Relíquias</h1> 

    <p>Neste espaço está em construção o futuro site do Armazém Café e Relíquias.</p> 
    <p>Parada obrigatória de todo visitante em Cunha.</p>

    <p lang="en-us">Here is under construction the future site of Armazém Café e Relíquias.</p>
    <p lang="en-us">An obligatory stop for every visitor in Cunha.</p>
  </body>
  
  </html>

Lembrando que lang é um atributo opcional em uma tag de texto. Você inclui se quiser mas é útil para o navegador ou para um programa leitor de tela para deficientes visuais, por exemplo.

Agora, para dar um exemplo de atributo obrigatório em uma tag, vamos introduzir uma nova tag extremamente importante no HTML. A tag <a>.

Este a vem de "anchor" e é justamente a tag responsável por criar links em um documento HTML. Veja como você poderia inserir um link clicável na sua página:


  <!DOCTYPE html>

  <html lang="pt-br">
    
  <head> 
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Exemplificando Links</title>
  </head>

  <body> 
    <h1>Armazém Café e Relíquias</h1> 

    <p>Neste espaço está em construção o futuro site do Armazém Café e Relíquias.</p> 
    <p>Parada obrigatória de todo visitante em Cunha.</p>

    <p lang="en-us">Here is under construction the future site of Armazém Café e Relíquias.</p>
    <p lang="en-us">An obligatory stop for every visitor in Cunha.</p>

    <p>
      <a href="https://viagemeturismo.abril.com.br/cidades/cunha/" target="_blank">
      Clique aqui para conhecer Cunha</a>.
    </p>
  </body>

  </html>

O navegador exibiria assim o código acima:

O conteúdo de uma tag <a> (o elemento entre a tag de abertura e a de fechamento) é um link que, se clicado, abre no navegador o endereço indicado no atributo href da tag. Portanto href é um atributo obrigatório em uma tag <a>.

Em um documento HTML, um link para o site do Armazém Café e Relíquias, poderia ser da seguinte forma:


  <a href="www.armazem.orgfree.com">O Armazém</a>

Este código faria O Armazém ser exibido como um link clicável para o site www.armazem.orgfree.com
