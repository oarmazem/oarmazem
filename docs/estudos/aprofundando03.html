Aprofundando Mais Um Pouco

Vamos continuar dissecando, aos pouquinhos, o código fonte que exibe aquela página no site http://www.armazem.orgfree.com

Aqui está ele novamente:


  <!DOCTYPE html>

  <html lang="pt-br">
    
  <head> 
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Armazém</title>
  </head>

  <body> 
    <h1>Armazém Café e Relíquias</h1> 
    <p>Neste espaço está em construção o futuro site do Armazém Café e Relíguias.</p> 
    <p>Parada obrigatória de todo visitante em Cunha</p>
  </body>

  </html>

Isto é um arquivo que você pode escrever e editar com qualquer editor de textos simples, como o Bloco de Notas. Um arquivo texto como este é o que o servidor envia para sua máquina e é interpretado por um navegador como o Google Chrome para renderizar a página.

Estes arquivos devem ser criados com a extensão .html e estarão lá, no servidor, esperando para serem solicitados pelos "browsers". Criar um site é basicamente escrever arquivos HTML como este e colocá-los em um servidor de internet.

Mas para escrever e editar arquivos HTML, não poderemos usar um editor como o WORD ou qualquer outro sofisticado. Pois estes irão inserir no arquivo códigos que o navegador não entende. Arquivos de linguagens como HTML são chamados de “plain text” ou texto simples. E contêm apenas códigos de caracteres e nada mais. Nós podemos usar um editor simples como o Bloco de Notas ou alguma IDE própria para escrever códigos de linguagens.

Mas voltemos a analisar o HTML acima.

Já vimos o que é cada tag neste código e o que faz. Porém vamos examina-las um pouco mais detalhadamente.

Há duas coisas mais que você pode descobrir prestando atenção no código acima:

  1. Nem toda tag tem uma correspondente tag de fechamento. A tag <meta> não tem. Não existe a tag </meta>
    2. .
  2. As tags podem possuir atributos. Note a tag <html> de abertura. Veja que coloquei no interior desta tag o atributo lang. <html lang="pt-br">

    3. O atributo vem sempre seguido por um sinal de igual e um valor entre aspas. Neste caso o atributo lang está informando ao navegador que o idioma que o documento HTML irá exibir é o Português do Brasil. Quando o Google Chrome pergunta se você quer traduzir um site, é para este atributo que ele está olhando para saber qual o idioma predominante na página.

    E perceba que as tags sem fechamento (como a tag meta) possuem apenas atributos mas não conteúdo. Conteúdo de uma tag é tudo que está entre a tag de abertura e a correspondente de fechamento. O título Armazém Café e Relíquias é o conteúdo da tag <h1>.

Já tudo que você vê abaixo é o conteúdo da tag <body>


  <h1>Armazém Café e Relíquias</h1>
  <p>Neste espaço está em contrução o futuro site do Armazém Café e Relíguias.</p>
  <p>Parada obrigatória de todo visitante em Cunha</p>

Como vê o conteúdo de uma tag pode incluir outras tags. É possível ter tag dentro de tag, dentro de tag, etc…

Por exemplo: dentro do <body> eu posso ter um título <h1> e dentro desse título dois parágrafos <p>.

Assim:

   
  <body>
    <h1>
      <p>Este é um título composto</p>
      <p>por dois parágrafos.</p>
    </h1>
  </body>
